PROJECT OVERVIEW
================================================================================
MY HOST BizMate is a comprehensive multi-agent AI system for vacation rental
property management. Built with React + Vite + Tailwind CSS, the application
provides property managers with an integrated platform for managing all aspects
of their vacation rental business.

ARCHITECTURE VERSION: v4.0 - Multi-Agent AI System

TECHNOLOGY STACK
================================================================================
- Frontend Framework: React 18 (functional components + hooks)
- Build Tool: Vite 6.0
- Styling: Tailwind CSS 3.4
- Icons: Lucide-react
- Version Control: Git
- Development Server: localhost:5178
